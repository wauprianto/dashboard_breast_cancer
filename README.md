Breast Cancer Diagnostic Dashboard 🧬
Dashboard interaktif untuk menganalisis dan memprediksi diagnosis tumor payudara menggunakan model Logistic Regression dan Random Forest. Proyek ini dibuat sebagai bagian dari studi Statistika untuk menerapkan teknik inferensi statistik dan machine learning.
🚀 Fitur Utama
 Analisis Eksploratif: Visualisasi distribusi data dengan grafik interaktif.
 Perbandingan Model: Membandingkan performa antara Logistic Regression (statistika klasik) dan Random Forest (machine learning modern).
 Evaluasi Model: Visualisasi ROC Curve dan Confusion Matrix untuk mengukur kualitas prediksi secara mendalam.
 Analisis Variabel: Visualisasi Feature Importance untuk mengetahui variabel mana yang paling berpengaruh terhadap diagnosis.
 Prediksi Real-time: Kalkulator interaktif menggunakan fungsi sigmoid untuk simulasi pasien baru.
 File Uploader: Mendukung unggah dataset kustom untuk pengujian model.
🛠️ Teknologi yang Digunakan
 Python: Bahasa pemrograman utama.
 Streamlit: Framework untuk membangun antarmuka dashboard.
 Scikit-Learn: Untuk implementasi model ML dan metrik evaluasi.
 Statsmodels: Untuk estimasi parameter statistik.
 Plotly & Seaborn: Untuk visualisasi data interaktif dan statis.
